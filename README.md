# 🎰 Roulette Table ASCII

## 🌟 Features

- **Interactive Gameplay**: Place bets on specific numbers, colors (Red/Black), or parity (Even/Odd).
- **ASCII Art Table**: A dynamically generated and colorized roulette table displayed directly in your terminal.
- **Real-time Balance**: Start with a virtual balance and try to win big!
- **Colorized Output**: Uses `colorama` to provide a visually appealing terminal experience.

## 🚀 Getting Started

### Prerequisites

You will need Python installed on your system. Additionally, you need the `colorama` library for the colored terminal output.

Install dependencies using pip:

```bash
pip install colorama
```

### Running the Game

To play the full interactive roulette game, run:

```bash
python roulettegame.py
```

### Other Files

- `Roulette with color.py`: A script that displays the colorized ASCII table without the game logic.
- `Roulette ASCII.py`: A simple script that displays the basic ASCII table in plain text.

## 🎮 How to Play

1.  **Set your bet amount**: Enter the amount of money you want to wager for the round.
2.  **Place your bets**: Enter numbers (0-36), colors (`Red`, `Black`), or parity (`Even`, `Odd`).
3.  **Spin the wheel**: Enter `Q` to finalize your bets and spin the wheel.
4.  **Check your winnings**: The game will announce the winning number and color, and update your balance accordingly.

## 🛠️ Built With

- **Python**: Core programming language.
- **Colorama**: For terminal styling and colors.
- **Random**: For simulating the roulette spin.
